XabslEditor

XabslEditor is a graphical editor for the Extensible Agent Behavior Specification Language XABSL (http://www.xabsl.de/). It is implemented in Java and should run on every platform that supports Java (Windows, Linux, Mac, etc.).

Shortcuts

General

ctrl + Wclose the active tab
ctrl + Oopen a file
ctrl + Fsearch in the current file
ctrl + shift + Fsearch in the project
ctrl + Ssave current file
ctrl + shift + Ssave current file as...
alt + Ccompile behavior
ctrl + Rrefresh graph
ctrl + Xclose XabslEditor

Editor Panel

ctrl + Spaceshow autocompletion
ctrl + left click on an optiongo to the declaration file of this option
